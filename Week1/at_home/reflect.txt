1. What is namespace? Explain its purpose.

    Namespace is a scope that defines a block of code. It prevents conflict of any variable with the same name.

2. Why are header files needed? Explain.
    
    Header files are needed for declaration of functions, variables, and structures.

3. Explain why you don’t compile header files.

    Header files do not have running codes, they only contain declarations.

4. Explain why you should never include .cpp files in another file.

    Including .cpp files in another file will resulting in you having two copies of the same code and may trigger errors.
